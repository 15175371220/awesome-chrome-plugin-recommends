    Tools

    Developer Tool

    • Resolution Test - An extension for developers to test web pages in different screen resolutions.
    • Octo Mate - List the navigation, file download, notification, etc. on the right side of the page.
    • Wappalyzer - Identify web technologies.
    • Clear Cache - Clear cache and browse data with the click of a button.
    • Proxy SwitchyOmega - Manage and switch multiple proxy settings quickly and easily.
    • WEB前端助手(FeHelper) - FE assistant.
    • classic-cache-killer - Automatically clear the browser cache before loading the page. It can be enabled/disabled with a single click of the mouse.
    • Window Resizer - Resize the browser window to simulate various screen resolutions.
    • eye-dropper - Eye Dropper is an open source extension that allows you to choose colors from web pages, color pickers and personal color history.
    • Axure RP Extension for Chrome - Allows an extension of the AXEURRP prototype to be viewed locally from Google Chrome.
    • JSON Viewer - Re-formatting data in JSON format to present JSON documents more intuitively in a tree view style.
    • Octotree - Generate GITHUB code tree browsing tool.
    • Postman Interceptor - Send a request triggered by the Postman Chrome app.
    • React Developer Tools - Add the React debugging tool to the Chrome Developer Tools.
    • Redux DevTools - Redux DevTools is used to debug state changes to the application.
    • Vue.js devtools - Chrome and Firefox DevTools extensions for debugging Vue.js applications.
    • User-Agent Switcher - Browse or add your own user agent using our predefined user agent.

    Search Tool

    • 右键搜 - Expand right-click menu with over 300 features.

    Production Tool

    • Checker Plus for Google Calendar™ - From the extension icon, you can get information about the meeting schedule and the date of todayl.
    • Auto refresh Page plugin - The best free page refresh plugin helps you automatically refresh the page at the right time and at the right time at the specified interval.
    • PDF Viewer - Uses HTML5 to display PDF files directly in the browser.
    • Favioli - Emoji favicons for the web
    • 划词翻译 - A simple but powerful translation extension.
    • Google access assistant - The simplest and easiest Google Access Assistant is tailored for Chrome Extension users.
    • Mainichi Learn Japanese each time you open a new tab.
    • Google 翻译 - The translated version can be easily viewed while browsing the web. Provided by the Google Translate team.
    • Google Keep Chrome - Save specific webpages, images, or quoted text to Google Keep with a single click!
    • 滴答清单 - Todo & 任务提醒 - Simple, easy-to-use to-do and task management tools to help you do everything efficiently, plan your time, and make life easier!
    • Clipbrd Beta - Synchronize the clipboard between all devices!
    • 云盘万能钥匙 - Automatically fill in the Baidu network disk to extract the password.
    • BaiduPan Explorer - Get Baidu network disk high speed download link.
    • OneTab - Save up to 95% of memory and reduce tab clutter.
    • IE Tab - The web page is displayed in the IE kernel in the tab. Fast, robust and reliable. This version is one of the most popular reasons.
    • 掘金 - Discover high-quality content daily for designers, programmers, and product managers.
    • 待办事项列表及任务管理器 - Todoist manages millions of tasks and is the best online to-do list. Works on Chrome, the web, iPhone, Android, Firefox, Outlook, and other platforms.
    • Evernote Web Clipper - Use the Evernote extension to save what you see on the web to your Evernote account.
    • Grammarly for Chrome - Strengthen your written communication, no matter where you type.
    • Adblock Plus - Adblock Plus is one of the world's most popular ad blocking software with over 500 million downloads.
    • Tampermonkey - Install Management Oil Monkey Script (GreaseMonkey).
    • 二维码(QR码)生成器 - The current page QR code can be generated, and the current page can be accessed by directly scanning the QR code on the mobile phone.
    • 简悦 - SimpRead - A Chrome extension that lets you instantly immersively read, similar to Safari's reading mode.
    • 迅雷下载支持 - Thunder download support allows you to quickly launch the Thunder client for download when you click on the download link in the Chrome browser.
    • Vimium - Keyboard streaming essential artifacts, you can use VIM to operate Chrome, Geek and keyboard operation controls.

    Extension Program

    • Undo Close Tab - Can restore the recently closed tab page.
    • arc-welder - A tool to help you use App Runtime for Chrome (beta), ARC testing and publishing Android apps to Chrome OS.

    Entertainment

    • 豆瓣FM-Hacker - Embedded in the web version of Douban FM (douban.fm) page, list the song information that Douban FM has played, and provide song downloads, more single loops, red hearts songs and batch download scripts.

    Photo

    • Image Downloader - Download the image on the web page.
    • Imagus - These images, HTML5 video/audio, and content albums are displayed directly on the pop-up view of the current page when the mouse pointer hovers over the link or thumbnail.
    • Fatkun - Find all the pictures on the current page, provide filters by resolution, links, etc., and download the images in batches.
    • 新浪微博图床 - Simple and easy to use Sina Weibo map bed, support selection / drag / paste upload pictures, and generate image address, HTML, UBB and Markdown formats, support browsing and delete history.

    Social And Communication

    • Google Mail Checker - Shows the number of unread messages in your Google Mail inbox.
    • Checker Plus for Gmail™ - Receive email notifications on your desktop without having to open Gmail or Inbox, easily view, listen to or delete messages, and support multiple accounts.

    News And Weather

    • Weather - The best way to view Weather in your browser. It's easier than looking outside!

    Shopping

    • 京价保 - 京东价保助手 - Beijing price insurance (Jing price treasure) is a multi-functional Jingdong assistant who automatically applies for Jingdong price protection for you, and automatically signs in to Beijing Jingdou and Gangwon.
    • 全网自动比价工具 - When browsing the product page, it automatically compares the lowest price of the same item in Taobao/Jingdong/Amazon/Dangdang/Suning/100 stores, and provides price history and word-of-mouth rating.

    Blog

    • Daily-Front-End-news - Front-end daily frontier information push - front end, Node, visualization, Python, tools, artificial intelligence, blockchain, etc.

    Sports

    • 365Scores - Live scores, news and real-time notifications of your favorite sports - NFL, MLB, NBA, NHL, NCAA, football, tennis and more.

    Accessibility

    • 关灯看视频 - Automatically dim the page as you watch the video, making you feel like you are watching a video in a movie theater. Available for YouTubeTM and other video sites.
    • Incognito-Filter - Let the specified website open in an "incognito window" to deal with sites that don't want to leave a trace.
    • download-manager - Manage your downloads in the top right of Chrome's toolbar.
